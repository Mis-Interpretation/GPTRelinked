## How to install
If you have UE5 with C++ working on your PC before, then you are fine. You should install visual studio build tool for C++ on your windows computer. Download the Installer: Go to the Official Microsoft Build Tools page and click "Download Build Tools."

Run the Installer: When the "Visual Studio Installer" opens, it will ask you what you want to install.

Select the Workload: Check the box that says "Desktop development with C++".

Confirm the Details (on the right side): Make sure the following are checked:

MSVC v143 (or the latest version)
Windows 10 SDK (or 11 SDK)
Click Install: This is a large download (about 1.5GB to 2GB) and will take some time.
